# Mock Interview Assets

## Default Interviewer Avatar

Place a `default-interviewer-avatar.png` image in this directory to serve as the default avatar for the SadTalker video generation.

**Requirements:**
- Filename: `default-interviewer-avatar.png`
- Format: PNG or JPG
- Recommended size: 256x256 pixels (or larger, will be resized by SadTalker)
- Should be a clear, front-facing portrait photo
- Good lighting and neutral background work best for SadTalker

**Example sources for avatar images:**
- Use a stock photo from sites like Unsplash or Pexels
- Generate an AI portrait using Stable Diffusion or DALL-E
- Use a professional headshot

The image will be publicly accessible at:
`https://your-worker-url/assets/default-interviewer-avatar.png`
